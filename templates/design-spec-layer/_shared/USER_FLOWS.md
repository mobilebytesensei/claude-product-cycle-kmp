### Implementation Details

#### Layer 1: SplashScreenViewModel

**Location**: `cmp-navigation/splash/SplashScreenViewModel.kt`

```kotlin
private fun determineNavigation(userData: UserData): SplashEvent? {
    return when {
        userData.firstTimeUser -> SplashEvent.NavigateToOnboarding
        !userData.isAuthenticated -> SplashEvent.NavigateToAuth
        else -> SplashEvent.NavigateToHome(...)  // Mood check delegated
    }
}
```

**Delay Times**:
- First-time users: 3.5 seconds (appreciate branding)
- Returning users: 2.5 seconds (faster access)

#### Layer 2: AuthenticatedNavbarNavigationViewModel

**Location**: `cmp-navigation/authenticatednavbar/AuthenticatedNavbarNavigationViewModel.kt`

```kotlin
init {
    checkActiveUserMood()  // Called on every authenticated session start

private fun checkActiveUserMood() {
    viewModelScope.launch {
        val activeUserMood = userMoodRepository.getActiveUserMood()
        if (activeUserMood == null) {
            sendEvent(AuthenticatedNavBarEvent.NavigateToMoodSelection)
        }
    }
}
```

**Key Behaviors**:
- Single source of truth for mood-based navigation
- Runs automatically when user enters authenticated area
- On error, defaults to mood selection (safe fallback)

## Flow 3: Mood Selection Flow

### Two Modes

| Mode | Trigger | Steps | Completion |
|------|---------|-------|------------|
| ADD | No active mood + authenticated | 4 steps | Navigate to Home |
| UPDATE | User manually changes mood | 3 steps | Pop back to previous |

### ADD Mode (First Time / No Mood)

```
+-------------+     +-------------+     +---------------+     +----------------+
| Onboarding  | --> | Select Mood | --> | Select        | --> | Movie Finder   |
| "Start"     |     | (grid)      |     | Emotion       |     | Engine         |
+-------------+     +-------------+     +---------------+     +-------+--------+
                                                                      |

                                                              +-------+--------+
                                                              |     Home       |
                                                              | (mood is set)  |
                                                              +----------------+
```

### UPDATE Mode (Change Mood)

```
+-------------+     +---------------+     +----------------+
| Select Mood | --> | Select        | --> | Movie Finder   |
| (grid)      |     | Emotion       |     | Engine         |
+-------------+     +---------------+     +-------+--------+
      ^                                           |
      |                                           v
+-----+------+                            +-------+--------+
|  Profile   |                            |   Pop Back     |
|  Settings  |<---------------------------| to Profile     |
+------------+                            +----------------+
```

### Key Behaviors

1. **Skip Button**:
   - ADD mode: Hidden (user must set mood for personalization)
   - UPDATE mode: Visible as "Cancel" (user can keep existing mood)

2. **Pre-selection**:
   - ADD mode: No pre-selection
   - UPDATE mode: Current mood and emotions pre-selected

3. **Completion**:
   - ADD mode: `setShowMood(false)` + Navigate to Home
   - UPDATE mode: Pop back to previous screen

## Error Handling

### Network Errors During Navigation

| Scenario | Fallback Behavior |
|----------|-------------------|
| Mood check fails | Navigate to Mood Selection (safe default) |
| Session check fails | Navigate to Sign In |
| User data unavailable | Navigate to Onboarding |

### Example Error Handling

```kotlin
private fun checkActiveUserMood() {
    viewModelScope.launch {
        try {
            val activeUserMood = userMoodRepository.getActiveUserMood()
            if (activeUserMood == null) {
                sendEvent(AuthenticatedNavBarEvent.NavigateToMoodSelection)
            }
        } catch (e: Exception) {
            logger.e(e) { "Error checking active user mood" }
            // On error, navigate to mood selection to be safe
            sendEvent(AuthenticatedNavBarEvent.NavigateToMoodSelection)
        }
    }
}
```
